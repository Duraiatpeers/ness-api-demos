Http Methods(Get, Post, Put, Delete,Patch,Options):
    - Safe
    - Idempotent


Status code:
1xx - Informational
2xx - Success
    - 200 - Success
    - 201 - Created
    - 204 - No data
3xx - Redirection
    - 302 - redirection
4xx - Client errors
    - 400 - Bad request
    - 401 - Unauthorized
    - 403 - Forbidden
    - 404 - Resource not found
    - 405 - Method not supported
    - 409 - Conflict
    - 415 - Mediatype not supported
5xx - Server error
    - 500 - Internal server error
